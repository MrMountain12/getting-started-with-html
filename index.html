NOTES ON PROGRAMMING:
Inline

Inline codes are codes that deal with the text lines, like (b) or (strong) (I'm using parentheses because if I use the actual signs, they won't show up)

Examples of inline are: (b) or (strong) for bold, (em) for italics, (a) and (img) for adding links and photos, and (br), which ends an existing line

Block
The other type of programming is Block, which forms an invisible box around the said text. The block htmls I've learned so far are (p),(form), and (div), which all start a new "paragraph"

Extra data
There are also different types of tags, like (form) or (br). They are called Void tags, because they contain no closing tags.
Computers are considered stupid because coding has to be exact for them to cooperate. If the code isn't correct, the code will not show up correctly
